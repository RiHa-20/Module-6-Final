What Modules 4-6 taught me about HTML and CSS

There was a lot to learn from Modules 4 through 6. It was fun to be able to learn how to build a website from the ground up with HTML and CSS. There are several semantic tags and CSS properties used in this project to make it interesting. There were a lot of different HTML tags and CSS properties used in this project that are important to it. The most important HTML tags used in this project are the semantic tags because they help divide the website into manageable parts and allow for different CSS properties to be applied to different portions of this website. Another extremely beneficial tag for this project was the map element because it allows an image to be used as a redirect link as it does here in my project. There are several CSS properties used in this project and all of them are important in making this site be able to be divided into parts so that content is easier to manage and the reader can see the parts of the webpage. One of the most important CSS properties used in this site is the "@media" rule, which allows this site to be responsive and be able to be read on multiple devices without having to have a scroll bar. Another really important CSS rule used in this site is the "@keyframes" property that allows this site to have different animations when the user switches pages. Additionally, the use of colors through CSS allows for there to be an easier seen separation between the different sections of content on this website. The CSS colors also allow users, when they mouse over a page link or collapsible, to see what will be opened and what the currently active page they are on is. Overall, there was a lot to learn about HTML and CSS that help to make this page what it is. Without CSS this website would not have easily seen separation making it harder for users to read.

There is a lot to learn from Module 4. Module 4 introduced us as a class to basic HTML. The most relevant topics discussed in module 4 that relates to this project was how to create an HTML skeleton, this helped to create the basic structure of every page on this site. Additionally, module 4 introduced the W3C validator which is a useful tool for anyone learning HTML to validate their site. There is a lot to learn from Module 5. Module 5 introduced us as a class to CSS and how it can be used to layout a website and change many of different aspects of a website. Additionally, module 5 introduced responsive web design which allows for a website to change its sizing and format when the sizing of the screen a website is viewed on changes. Responsive Web Design (RWD) and CSS are used in this site to control several aspects including colors, sizings, and used to control several aspects when the device screen size gets smaller. There is a lot to learn from Module 6. Module 6 goes into further advanced HTML and CSS practices, which helps to make a website look nicer and more advanced. Additonally, module 6 talkes about semantic tags furtherand shows how they give meaning to the code and make a website structured better.
